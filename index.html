Exercício de fixação

Tag <title>
É a tag responsável por dar o título a página, o nome que aparece visível na aba.
Tag <body>
Tag que representa o corpo do documento, onde todo o conteúdo visível ao usuário será inserido
Tag <p>
Essa tag determina um parágrafo
Tag <b>
Deixa o texto em negrito
Tag <i>
Deixa o texto em itálico
Tag <br>
Essa tag não necessita de fechamento, e serve para concluir o texto
Tag <img>
Essa tag não precisa de um fechamento, e é utilizada para inserir a imagem
Deve ser usado junto com o atributo scr
Tag <ul>
Tag que determina uma lista, cuja ordem dos itens não é ordenada
Tag <ol>
Tag que determina uma lista, cuja ordem dos itens é ordenada
Tag <il>
Tag que caracteriza os itens da lista, seja ela ordenada ou não
